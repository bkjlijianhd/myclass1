
# npm 使用

```
npm install   xxxx  安装(jquery)
npm installl  xxx@version  (指定版本安装)
npm uninstall   xxxx       (卸载之前安装)

npm  install   -g  xxx
npm  uninstall   -g  xxx

npm update  xxx (更新...)


npm install xxx@latest //最新版
npm ls (查看所有模块)
npm ls  -g (查看全局模块)


npm --help 帮助文档


npm  init  --yes //产生一个package.json文件

npm install jquery  --save  // --save 将jquery作为项目依赖填写到package.json
npm install jquery  -S  // -S 将jquery作为项目依赖填写到package.json
npm install xxx  --save-dev  //将xxx作为开发工具类的模块依赖

npm install   //直接安装package.json中的依赖模块

```
